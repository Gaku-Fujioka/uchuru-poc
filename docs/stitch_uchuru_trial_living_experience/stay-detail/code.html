運営体制について

本滞在プログラムは、役割を分担して運営しております。

uchuru 担当

生活体験ミッションの提供、滞在中のUXサポート、コミュニティ連携。

運営事業者 担当

宿泊予約の管理、鍵の受け渡し、清掃業務等の施設管理全般。

* 初期PoCでは、契約・決済・鍵管理・清掃は各事業者が行います。uchuruは、送客導線、滞在UX、生活情報オリエンテーション、アンケート分析に絞って検証します。
